E-Commerce Platform (MERN)

Full-stack eCommerce application with user and admin functionalities.


Features
- User authentication (JWT)
- Product listing & cart management
- Admin dashboard for product management
- Order handling system


Tech Stack
- Frontend: React.js, Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB


Project Structure
- `/client` → User website
- `/admin` → Admin panel
- `/server` → Backend API

  
Run Locally

1. Start Backend
```bash
cd server
npm install
npm run dev
```

2. Start Client
```bash
cd client
npm install
npm run dev
```


3. Start Admin Panel
```bash
cd admin
npm install
npm run dev
```
